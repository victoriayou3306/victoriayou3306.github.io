# assets/

Put your files here and reference them in the HTML.

## Files to add:
- `profile-photo.jpg` — your profile photo (referenced in index.html and about.html)
- `resume.pdf` — your resume (linked in index.html and about.html)

## How to reference them in HTML:
```html
<img src="assets/profile-photo.jpg" alt="My photo" />
<a href="assets/resume.pdf">Resume</a>
```
